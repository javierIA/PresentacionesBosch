# Crear un DataFrame ðŸ› 

```python
import pandas as pd

data = {'Name': ['John', 'Anna', 'Peter'],
        'Age': [28, 24, 33],
        'City': ['New York', 'Paris', 'Berlin']}

df = pd.DataFrame(data)
```
